# 知识点随记

## 6 人間の能力と運転

- 认知 -> 判断 -> 操作
  - 事故占比 85 / 15 / 5
- 反应时间
  - 大概1s = 反射时间 + 移动到刹车的时间 + 踩下刹车的时间
  - 对应“空走距離”
  - 反应时长会根据外界环境变化
  - 限界がある
  - 正确对应
    - 提前控制速度
    - 提前把脚放到刹车上
- 反应时间内的行进距离
  - 40km/h => 约10m
  - 60km/h => 约20m
- 主要信息来源是 —— 视觉
- 静止視力
  - 双方都是静止的，注视一个点
  - 特点：注视点以外的部分会难以分辨
  - 行驶中，要一直保持看各处而不是盯着一个点
- 動体視力
  - 有一个是运动的，就对应动态视力
  - 速度越快，能看清楚的东西越少
  - 年龄大了，动态视力会下降
- 视野
  - 单眼160度；双眼200度；左右35度是可以分辨颜色的区间
  - 需要看红绿灯的时候，尽量使用视野的中心区域<=水平高度稍微往上一点看
  - 速度越快，视野越窄
- 順応
  - 暗順応——明到暗（耗时更长）
  - 明順応——暗到明
  - 进入隧道、夜晚行车需要注意
- 速度
  - 高速下来之后判断失误
  - 对于对向来车速度的距离判断失误
  - 时不时地确认速度
- 常见的判断失误
  - 相同距离，觉得大车比小车近
  - 高度道路，觉得自己没开那么快
  - 晚上因为没有参照物，觉得自己没开那么快
  - 夜间，只通过对方车灯判断的时候，对方可能比想象的要近、要快
  - 长下坡的时候，坡度变化以为开始上坡了
- 容易引起失误的因素
  - 疲劳
  - 過労運転 —— 2小时休息一次
  - 饮酒
  - 药物

## 7 車に働く自然の力と運転

- 惯性
- 摩擦力
  - 受气温影响
  - 摩擦力太小，难以启动/难以停下
- 离心力
  - 指向外侧
  - 离心力 > 摩擦力，会发生侧滑
  - 离心力本质上是惯性的表现，质量越大，离心力越大
- 停止距离
  - 车承载物up，距离up
  - 速度up，距离up
  - 湿滑地面，停止距离大概是干燥路面的2倍
- 制动距离
  - 是速度的平方
  - 50km/h =>  18m
  - 100km/h => 84m
- “engine brake”
  - 松开油门后，引擎转速降低，降低车轮转速
  - 在自动挡车上不是那么管用
- 急刹车
  - 保持方向向前
  - 防止刹车锁死（会侧滑、并且摩擦力减小延长停止距离）
- ABS
  - 急刹的时候，电脑控制，转换成多次密集点刹
  - 在特定条件下，比如低速/砂石路，可能会制动距离增加
- 上下坡
  - 突然过度降档减速可能导致追尾
  - 选择和坡度符合的降档，L/2
  - 上坡优先
  - 如果有滑坡危险，山体侧的线停车
- 制动衰退 vs 气阻现象
  - fade（制动衰退）
    - 摩擦部分过热的问题
  - vapour lock（气阻现象）
    - 液压系统过热
    - 更危险
- 盘式刹车 vs 鼓式刹车
  - 盘式	
    - 散热好
    - 不容易热衰减
    - 现代汽车前轮基本都是盘刹，高级车四轮都是盘刹
  - 鼓式
    - 里面有刹车蹄向外撑开摩擦
    - 便宜、耐用、手刹结构容易做
    - 小车后轮，轻型车还常见
    - 散热差，连续刹车容易过热
- 碰撞
  - 冲击力决定了事故的大小
  - 和速度、质量关系很大；越硬的物体，短时间内累计的冲击力越大
  - 60km/h的速度撞上墙壁：14m高(五层楼)落下的冲击力
- 交通公害
  - 噪音
  - gas
- 摩托车
  - 离心力和重力的合力指向轮胎的时候，是最稳定的
  - 上下坡的时候需要移动人的重心，上坡站起来，下坡腿登直
  - 自动挡 vs 手动挡
    - 自动挡车体长、轮子小、车体重
    - 自动带拐小弯困难，容易受到路面颠簸的影响
    - 无级变速装置的情况下，因为变速缓慢，很容易因为行驶速度低失去平衡<=同时使用刹车和油门来掌握平衡
  - 手动挡摩托车驾驶姿势
    - 脚尖朝向前方，膝盖夹住车体
    - 手腕放松，像推车一样
  - 选车原则
    - 跨过摩托车，两脚可以着地
    - 有能力推着走8字路线
    - 如果需要两个人乘坐，确认后座空余量
  - 视角
    - 容易下意识集中在左侧
    - 要多注意右侧和对向右侧来车
  - 转弯
    - 转弯的时候不是转动方向把手，而是靠身体倾斜来转弯
    - 转弯过程中不要松开离合，用油门来控制速度（松开离合，会导致摩托车失去动力牵引，失去稳定性）
  - 刹车
    - 同时使用前后刹车
    - 因为刹车的时候，大部分重量会放在前方车轮上
      - 干燥路面，前轮刹车多用力
      - 湿滑路面，后轮刹车多用力
    - 一级一级降档；分次刹车，防止侧滑
  - 二人同乘
    - 服装和头盔
    - 乘车途中很难交流，提前定好信号
    - 乘坐者要随着开车者的姿势，不要有多余的动作
    - 膝盖夹着驾驶员的盆骨
    - 两人座位保持合适距离
    - 一人习惯后 => 学习两人乘坐的要点
  - 驾照规定
    - 单人驾照 => 一年后，可以两人在普通道路上 => 三年后，可以两人在高速道路上
