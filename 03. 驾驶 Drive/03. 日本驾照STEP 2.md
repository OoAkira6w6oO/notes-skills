# 知识点随记

## 6 人間の能力と運転

- 认知 -> 判断 -> 操作
  - 事故占比 85 / 15 / 5
- 反应时间
  - 大概1s = 反射时间 + 移动到刹车的时间 + 踩下刹车的时间
  - 对应“空走距離”
  - 反应时长会根据外界环境变化
  - 限界がある
  - 正确对应
    - 提前控制速度
    - 提前把脚放到刹车上
- 反应时间内的行进距离
  - 40km/h => 约10m
  - 60km/h => 约20m
- 主要信息来源是 —— 视觉
- 静止視力
  - 双方都是静止的，注视一个点
  - 特点：注视点以外的部分会难以分辨
  - 行驶中，要一直保持看各处而不是盯着一个点
- 動体視力
  - 有一个是运动的，就对应动态视力
  - 速度越快，能看清楚的东西越少
  - 年龄大了，动态视力会下降
- 视野
  - 单眼160度；双眼200度；左右35度是可以分辨颜色的区间
  - 需要看红绿灯的时候，尽量使用视野的中心区域<=水平高度稍微往上一点看
  - 速度越快，视野越窄
- 順応
  - 暗順応——明到暗（耗时更长）
  - 明順応——暗到明
  - 进入隧道、夜晚行车需要注意
- 速度
  - 高速下来之后判断失误
  - 对于对向来车速度的距离判断失误
  - 时不时地确认速度
- 常见的判断失误
  - 相同距离，觉得大车比小车近
  - 高度道路，觉得自己没开那么快
  - 晚上因为没有参照物，觉得自己没开那么快
  - 夜间，只通过对方车灯判断的时候，对方可能比想象的要近、要快
  - 长下坡的时候，坡度变化以为开始上坡了
- 容易引起失误的因素
  - 疲劳
  - 過労運転 —— 2小时休息一次
  - 饮酒
  - 药物

## 7 車に働く自然の力と運転

- 惯性
- 摩擦力
  - 受气温影响
  - 摩擦力太小，难以启动/难以停下
- 离心力
  - 指向外侧
  - 离心力 > 摩擦力，会发生侧滑
  - 离心力本质上是惯性的表现，质量越大，离心力越大
- 停止距离
  - 车承载物up，距离up
  - 速度up，距离up
  - 湿滑地面，停止距离大概是干燥路面的2倍
- 制动距离
  - 是速度的平方
  - 50km/h =>  18m
  - 100km/h => 84m
- “engine brake”
  - 松开油门后，引擎转速降低，降低车轮转速
  - 在自动挡车上不是那么管用
- 急刹车
  - 保持方向向前
  - 防止刹车锁死（会侧滑、并且摩擦力减小延长停止距离）
- ABS
  - 急刹的时候，电脑控制，转换成多次密集点刹
  - 在特定条件下，比如低速/砂石路，可能会制动距离增加
- 上下坡
  - 突然过度降档减速可能导致追尾
  - 选择和坡度符合的降档，L/2
  - 上坡优先
  - 如果有滑坡危险，山体侧的线停车
- 制动衰退 vs 气阻现象
  - fade（制动衰退）
    - 摩擦部分过热的问题
  - vapour lock（气阻现象）
    - 液压系统过热
    - 更危险
- 盘式刹车 vs 鼓式刹车
  - 盘式	
    - 散热好
    - 不容易热衰减
    - 现代汽车前轮基本都是盘刹，高级车四轮都是盘刹
  - 鼓式
    - 里面有刹车蹄向外撑开摩擦
    - 便宜、耐用、手刹结构容易做
    - 小车后轮，轻型车还常见
    - 散热差，连续刹车容易过热
- 碰撞
  - 冲击力决定了事故的大小
  - 和速度、质量关系很大；越硬的物体，短时间内累计的冲击力越大
  - 60km/h的速度撞上墙壁：14m高(五层楼)落下的冲击力
- 交通公害
  - 噪音
  - gas
- 摩托车
  - 离心力和重力的合力指向轮胎的时候，是最稳定的
  - 上下坡的时候需要移动人的重心，上坡站起来，下坡腿登直
  - 自动挡 vs 手动挡
    - 自动挡车体长、轮子小、车体重
    - 自动带拐小弯困难，容易受到路面颠簸的影响
    - 无级变速装置的情况下，因为变速缓慢，很容易因为行驶速度低失去平衡<=同时使用刹车和油门来掌握平衡
  - 手动挡摩托车驾驶姿势
    - 脚尖朝向前方，膝盖夹住车体
    - 手腕放松，像推车一样
  - 选车原则
    - 跨过摩托车，两脚可以着地
    - 有能力推着走8字路线
    - 如果需要两个人乘坐，确认后座空余量
  - 视角
    - 容易下意识集中在左侧
    - 要多注意右侧和对向右侧来车
  - 转弯
    - 转弯的时候不是转动方向把手，而是靠身体倾斜来转弯
    - 转弯过程中不要松开离合，用油门来控制速度（松开离合，会导致摩托车失去动力牵引，失去稳定性）
  - 刹车
    - 同时使用前后刹车
    - 因为刹车的时候，大部分重量会放在前方车轮上
      - 干燥路面，前轮刹车多用力
      - 湿滑路面，后轮刹车多用力
    - 一级一级降档；分次刹车，防止侧滑
  - 二人同乘
    - 服装和头盔
    - 乘车途中很难交流，提前定好信号
    - 乘坐者要随着开车者的姿势，不要有多余的动作
    - 膝盖夹着驾驶员的盆骨
    - 两人座位保持合适距离
    - 一人习惯后 => 学习两人乘坐的要点
  - 驾照规定
    - 单人驾照 => 一年后，可以两人在普通道路上 => 三年后，可以两人在高速道路上

## 10 自動車の保守管理

- 日常需要检查什么
  - 窗户
    - 如果有油膜，可能会夜间反光，需要注意
  - 各个灯光是否好用
  - 雨刮器
    - 每年需要确认一下橡胶的情况，需要时进行更换
- 摩托车日常检查
  - 车灯
  - 各个部分的润滑
  - 车链条的状态
  - 螺丝情况
- 电瓶液
  - 硫酸约 30%～40%
  - 需要补充的时候，补充的是“蒸馏水”
- 非常信号用具
  - 红光手电筒
  - 发烟筒
  - 三角停止表、停止表示灯
  - 急救包、备胎、牵引绳、电瓶线、灭火器
- 交换备胎
  - 尽可能在道路外面，如果没办法可以在道路左侧；放下行李，在平坦坚硬的道路上进行
  - 提前放好三角警示板，
  - 备胎
    - 一般比标准空气压大，直径偏小
    - 难以承受100km/h以上速度或长距离乘车
    - 不能用链条
  - 步骤
    - 放置挡轮块：要放在“不会被抬起”的轮子上，最好放在对角线方向的轮子附近
    - 卸掉轮胎盖子，拧松螺丝
    - 使用千斤顶架起车
    - 卸下螺丝，交换备胎，放回螺丝
    - 放下千斤顶，拧紧螺丝（对角线五角星顺序）
- 轮胎换位（タイヤローテーション）
  - 每 5000～10000 km
  - 定期点検時
  - 前轮和后轮进行交换
  - 前轮更容易磨损
- 轮胎链条
  - 选择合适的尺寸，安装在驱动轮上
  - 安装好链条后，安装防滑链固定橡胶圈（部分链条不使用）
  - 注意链条钩子有刺的方向朝向外侧；Lock翻转一圈锁住
- FF是前驱，FR是后驱
  - 缩写：发动机位置（Engine）＋ 驱动轮（Drive）
  - FF = Front-engine Front-drive
  - FR = Front-engine Rear-drive
  - MR = Mid-engine Rear-drive 中置后驱
  - RR = Rear-engine Rear-drive 后置后驱
  - 4WD / AWD = Four-wheel drive 四驱
- 日常点検
  - 自分自身の責任
  - 日頃に行う
  - 適切な時期
  - 事業用の自動車だと、1日一回（载人/载货/出租用车）
  - 租车的时候，需要确认车的状况
- 定期点検
  - （ブ）ブレーキ 刹车
    - 踩起来做动范围是否有问题，来判断是否有刹车液漏液等问题
    - 刹车是否好用，在日常的低速运行中确认
  - （タ）タイヤ
    - 轮胎的气压一般在车门附近会写着标准的气压，如果需要上高速的时候一般会让气压+0.1，可以在加油站让帮忙处理
  - （と）灯火
  - 燃料
- 机油
  - 发动机里面有很多金属零件高速运动
  - 发动机内部油道，润滑，“别磨坏”
  - 如果引擎过热，应该保持发动机做动，等待温度降下来
- 风扇皮带／发动机皮带
  - 发动机转动时，通过这根皮带，带动其他设备工作
  - 驱动外围设备
- 正时皮带
  - 让发动机“动作同步”，活塞运动 和 气门开关
  - 现代多用链条
- 空气制动
  - 用压缩空气来刹车的系统
  - 大卡车，巴士，拖车
  - bus启动的时候的声音就是因为这个放气声音
  - 检查储气罐时，要打开排水阀（ドレンコック），确认储气罐里有没有积水
- 轮胎为什么是黑色的
  - 混入了26%左右的黑色的碳粉
  - 也有混入二氧化硅做成彩色轮胎的，但是性能不如碳粉好

## 11 駐車と停車

- 駐車
  - 5分钟以上的行李装卸
  - 等人
  - 没有办法立刻启动
- 停車
  - 和已经在的人汇合
  - 5分钟以内的行李装卸
  - 可以立刻启动
- 禁止地点
  - 駐停車
    - 有标志的地方
    - 轨道内
    - 坡顶
    - 大的上坡和下坡途中（上下都不可以！）
    - 隧道内
    - 交叉路口5m内（一台车的长度）
    - 拐角5m内（一台车的长度）
    - 人行横道前后5m内
    - 铁路交叉路口10m内
    - 安全地带10m内
    - 公交站牌10m半径（运行时间内）
  - 駐車
    - 火灾报警器1m半径
    - **车库出入口3m内**（刚好够一台车探头观察 + 基本转向安全空间，低速，省2m）
    - 道路施工范围5m内
    - 消防通道、消防设施5m内
    - 有特殊情况的时候可以向警察局提出申请
  - 無余地駐車
    - 道路太窄，一旦停车无法过车的时候不能长时间停车
    - 一般认为剩余空间时3.5m（一辆车的宽度），有时候标志会写明需要留下的余地
    - 搬家、救护车等例外
  - 標章者専用
    - 年龄大的、听觉障碍、孕妇或产后8周内，可以向公安局提出申请，取得标识
- 停车原则
  - 靠左停车，靠线/车道/马路牙子，尽可能离路边0.75m以上（轮椅宽度为0.7m）
  - 高速中，选择应急车道
  - 已经别人听了，不能再停一层
  - 如果规定了车头方向，需要遵守
  - 夜间停车/视野不好的地方
    - 打开双闪、停车灯、尾灯
    - 放置警示牌
  - 不可以连续12小时以上/夜间8小时以上
  - 如果自助停车有小票，需要贴在车窗前侧
- 违规停车的处罚
  - 拖走，罚款
  - 30天内支付罚款
  - 罚款票只能责任人揭下
  - 超时不交罚款，会影响下次车检
  - 在一定时间内违反的次数过多的话，会暂时吊销驾照
- 停车时的档位
  - 手动挡——下坡挂倒档，上坡挂低挡（都要往坡顶方向go，反自然作用力）
  - 自动挡——P
- 坡道停车时的轮胎方向
  - 自动车	
    - 上坡向右，下坡向左
    - 原则——溜车的时候往墙上撞
  - 摩托车
    - 尽量选择平坦路面
    - 挂一挡 —— 发动机 → 变速箱 → 后轮 = “机械锁住”
    - 把把手打到最左 —— 让车“自然向侧撑压住”，车重本来就在左侧侧撑
      - 这样侧翻的时候，刹车依旧好用……
- 车库
  - 住所附近2km，没有车库不能买车
  - 车库需要在警察局登记 “保管場所証明書”
  - 如果保管场所有变更，需要15天内去警察局登记
- 注意
  - 公交站不能 駐停車
  - 消防栓附近5m内，不可以 駐車
