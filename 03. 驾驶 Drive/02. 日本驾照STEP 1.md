# 知识点随记

- 作为牵引车的摩托车、sidecar，即使关闭引擎推着，也不能算步行者
- 徐行 = 10km/h 一下，停止距离1m以内，随时能停下
- 急坡 = 10%（6度）以上

## MORAL モラル

- 儿童座椅：6岁以下
- 责任 = 刑事 + 行政 + 民事
- 2瓶啤酒 = 6个小时持续
- 2小时休息一次
- 空气气囊，30km/h以上起效
- 需要带 = 驾照 + 检查证 + 保险证 + 备品 + mark

## 信号

- 看对面的信号灯
  - 不要看着红灯倒计时启动，看好路况 + 绿灯

- 摩托车、自行车：二段階右折
  - 看标识
  - 同时满足“多车道” + “有红绿灯”时必须
  - 期间需要打右转向灯
  - 摩托车，如果是非常简单的路口，没有信号灯的，要和车一样右转（小回り）
- 已经左转、右转到一半了，看到红灯/黄灯，需要尽快离开交叉路口
- 方向键
  - 红灯 + 黄色的方向键 —— 给**路面电车**看的
  - 红灯 + 绿色的方向键 —— 车可以走指定方向
- 小型车、自行车 —— 可以走人行道，过马路看人行道的灯
- 警察手势
  - 举手 = 准备变灯
  - 警察指令 > 信号灯指令

- 停止位置
  - 能看到红绿灯的位置 / 警察的1m前

- 停止线
  - 有时可能有两条，靠前的是给摩托车的（因为汽车死角的缘故，所以这么设计）

- 左折可（白底蓝色向左箭头） —— 即使红灯 / 黄灯，如果确认安全也可以从指定地方通过左转
- 灯一直慢闪烁
  - 红灯 —— **必须停止一次**
  - 黄灯闪烁 —— 确认安全可以缓慢通过

## 标识 & 标示

- 标识 = 本标识 + 辅助标识（天上立的标）
  - 本标识 = 规制（禁止类） + 指示 + **警戒** + 案内
- 标示 = 规制 + 指示 （地上画的线）
- 如果有数字的，数字本身的重量是ok的
- 最高速度 —— 车的，如果原付这种本身就限速的，那应该走本身的限速（30km/h）
- 標章車 = 高龄者等
- 鸣笛 / 区间鸣笛
- **安全地带 （ 向下的箭头）**
- 白底红色箭头 ——  区间
- 白色实线 —— 不可以超车，可以并道
- 方框 —— 停止禁止部分（电车可能在这附近停车的原因）
- 路侧带
  - 白色实线
    - 给人和自行车走的
    - 如果很宽的话可以停车
  - 白色实线 + 白色虚线（駐車停車禁止）
    - 给人和自行车走的
    - 不可以停车
  - 双白实线
    - 只有人能走
    - 不可以停车
- 白色斜线
  - 黄色：禁止进入
  - 白色：禁止停车


## 通行道路

- 左侧通行
- 通行原则
  - 道路 = 步道 + 车道
  - 车辆原则走车道
  - **特殊出入口可以跨步道，但是必须！一时停止！**
  - **回家的时候走步道 —— 必须徐行**
- 车道外侧线：有马路牙子的时候；路侧带：没有马路牙子的时候
- 车道
  - 单车道：靠左行驶；如果没有划线，离侧面最少50cm（建筑问题不牢固&自行车等通行）
  - 双车道：左车道行驶，右车道超车用
  - 多车道：最右侧车道超车用，从左到右，车速up

## 十字路口

- 最容易出事故的地方 —— 交差点
- 并道/转向前提前靠边
  - 靠左：防止自行车和摩托出现在死角
  - 靠右：让后方直行车辆可以通过
- 转向灯
  - 并道提前3s
  - 转向提前30m
  - 实操一般3s+30m

- 右转优先级最低，需要看别人的情况
- 有急救车等需要让路的时候，可以违反车道
- 进入加油站
  - 左拐：先靠边再左拐
  - 右拐：靠边，到了附近打直角右拐
  - 如果经过步道，必须一时停止
- 前面的车要变道的时候，不可以妨碍他
- 优先道路 —— 有划线 / 有标识
  - **如果都一样，注意看左侧车**
- 环岛
  - 顺时针（右）行驶 —— 驾驶员靠近中心 → 更容易观察环岛内车辆
  - must 缓慢行驶
  - **“外側”行驶** —— 方便驶出
- 踏切
  - 没有信号灯，必须一时停止，开窗户看、听 
  - 开始亮灯/下降，就不可以通过了
  - 万一的时候可以撞杆子
  - 防止轮子掉落，稍微靠中间
  - 出事儿，按按钮、放烟筒、烧东西
- 警车
  - 车：白色 => 黑白色
  - 摩托：红色 => 白色
- **安全地带 / 立ち入り禁止部分**
  - **无论什么时候都不能进**
- 有需要获得许可的，需要警察的许可 

## 速度

- 規制速度
- 法定速度
  - 原付 = 30km/h
  - 自动车 = 60km/h
  - 急救车 = 80km/h
  - 拖车 = 30km/h（根据情况25～40）
- 停止距离 = 空走距离 + 制动距离
  - 速度 x 2，制动距离 x 4
  - 40km/h => 22m
  - 60km/h => 44m
- 保持车距
  - 车的驾驶位置越高，人的感觉上前后距离会变短
  - **附近有大型车辆，增大车距**
  - 隧道，增大车距
    - 豆知识：隧道，一般是黄色钠灯；穿透能力强，不容易吸引飞虫；入口处更亮
- 安全车距
  - 80km/h以下：运行速度 - 15
  - 80km/h以上：安全距离 = 运行速度

- 刹车
  - 慢慢踩
  - 点刹
    - 天气路况不好
    - 防止刹车片过热失灵
  - 主要利用油门控制速度，减少急刹
- ABS
  - 防抱死制动系统
  - 在你紧急刹车时，防止车轮“锁死”；ABS 会通过快速“点刹”（每秒多次自动松开再制动），让车轮保持“将停未停”的状态，从而提高安全性
  - 缺点：可能因为路况情况，延长刹车距离
- 摩托车刹车
  - **逐级降低档位**，掌握好方向
  - 前后刹车一起用，路况好前轮刹车加力，路况不好后轮加力
  - 多次刹车
- 缓慢行驶
  - 10km/h
  - 能立刻停下来（1m以内）
  - 有标识 / 坡顶 / 下坡 / 视野不好 / 急转弯
  - 横跨步道 / 出路口 / 电车附近 / 拐弯 / 水坑 / 交叉路是优先道路
- 自动挡
  - 长下坡
    - 挂2 （缓坡）/L1（急坡），不踩刹车也不踩油门
    - D到L换挡，需要一档一档换
  - 停车
    - 手刹 +  空档
  - 完全停车的时候才能放P档
- 自动挡摩托车
  - 一般车长更长、车轮更小
  - 转向更困难
  - 膝盖夹不住
  - 发车类似坡起：后轮刹车的同时上油，慢慢松刹车
- CVT
  - Continuously Variable Transmission / 无段变速装置 / 无级变速器
  - 两个可变直径的滑轮，通过改变滑轮直径，丝滑变速
- ASV
  - 先进安全车辆（Advanced Safety Vehicle）
  - 一整套安全辅助系统的总称
  - 自动刹车 / 车道偏离预警 / 自适应巡航，保持安全距离 / 盲区监测 / 误踩油门抑制

## 步行者

- 安全距离 = 1 ~ 1.5 m
- 安全地带附近
  - 如果有人，必须减速行驶
  - 没有人可以安全通过
  - 路面电车
    - 原则：停在车后方等人都下车结束
    - **安全地带 + 路面电车：必须减速行驶；没有安全地带：停车等候**
- 人行横道30m范围内禁止超车
  - 如果有车已经停了，也应该停止
- 附近有学生bus —— 减速慢行
- 特殊标识
  - 特点
    - 前后都要贴
  - 新手、听力障碍 —— 必须有标识
  - 70岁以上 —— 红叶
- 有问题的车辆
  - 噪音、危险行为、故障、没有消音器
  - 15天以内整改，6个月 / 30w处罚

## 安全确认

- 前方、后方、侧方，死角（两侧安全柱附近）需要目视
- 安全确认 => 打信号 => 再次确认 => 行动
- 分两阶段开关车门，**如果右侧不好下车，翻到左侧下车**
- 停车的时候，尽量车屁股朝里，因为出发的时候更容易发生事故；倒车的时候找人帮忙看着
- 鸣笛区间
  - 在经过指定地点的时候鸣笛
    - 拐弯
    - 坡顶
    - 交叉点

## 变道

- 禁止黄线变道
  - 如果有急救车 / 工事的时候例外
- 转向灯
  - 每分钟60～120次，基本70～90次
  - 人类紧张的时候的心跳次数
- 注意区分专用 / 优先
  - **拐弯的时候，可以跑到其他专用车道上**

## 急救车

- 交叉口附近：快速离开交叉路口，靠边停车
- 非交叉口附近：一般道路，左侧靠边；单向路，右侧靠边，靠边后可继续行驶
- 总结 —— 有路口必须停车，没有路口可以靠边走

## 超车

- 不可以超车的情况
  - 二重超车 = 前面的车在超车 & 被超车的是自动车
  - 前方车辆变道
  - 有对向来车
  - 后面的车正打算超自己的时候
- 不可以超车的地点
  - 拐角
  - **坡顶**
  - 急下坡
  - **单车道隧道**
  - 交通路口30m之内不可以变道超车
- 标志区分
  - 不可以越线超车（一般）
  - 不可以超车（有字）
- 白色实线 —— 不可以越线超车
- 哪一侧超车
  - 一般来讲是右侧
  - 只有前车要右拐已经靠右的时候，可以左侧超车
  - **超电车的时候，左侧超车**
  - 被超车的时候要让让

## 过车

- 保持安全距离，看左右情况
- 前方有障碍物 —— 提前减速/停车
- 上下坡错车
  - 上坡优先
  - 有等待区，靠近等待区的让
  - 有悬崖，靠山的一侧让

## 驾照

- 16岁可以考摩托车驾照，18岁可以考自动车驾照
- **仮免許 —— 6个月有效**
  - 普通学习
  - 驾照的车种升级
  - 驾照的类型（如载客）升级
- 仮免許 副驾驶要求
  - **三年以上**驾驶经验 / 有第二种驾照 / 专业教练
  - 车**前后**必须写明 练习车
- 有效期间
  - 第一次：第三个生日后的一个月
  - 续期：根据情况5/4/3年
  - 优秀：五年没有违章
  - 国际驾照：一年有效
- 更新驾照
  - 可以提前两个月更新，最晚到生日后一个月
  - 异地更新：没有违章的话可以；提前两个月到一个月的期间内
  - 提前很久的更新可以说明具体情况更新
  - 每次更新的时候要听一次讲座 / 更新的六个月内听一次讲座，根据驾驶历史30～60～120分
    - 70岁：六个月内
    - 75岁：特殊检查
- 过期更新
  - 6个月以内：讲座 + 测试
  - 1年： 测试 后 可以获得 仮免許 
  - 特殊情况的3年以内：讲座+测试
- 新手期间（1年以内）
  - 2次违章，3分以上 —— 有一次机会参加讲习和考试，一个月内完成

# 刷题随记

- 原付 —— 只能驾驶员乘坐，不能载人
- 图标
  - **向下箭头：安全地带**
  - 横线：禁止进入
  - 斜线：禁止通行
  - 蓝底红色双叉：駐停車禁止；单叉：駐車禁止
  - 重量限制：**车辆整体**的重量
  - 其他车的专用道路：**原付、轻型车辆、自行车可以走；并且也不需要礼让**
  - 進行方向別通行区分（方的） / 指定方向外進行禁止（圆的）
  - 白色斜线：安全地带/障碍物
  - 白底红色【斜】箭头：指示方向
- に優先して —— 比……优先
- 闪灯
  - 红灯闪烁：必须一時停止
  - 黄灯闪烁：没有缓行的义务
- 超车
  - 即使是禁止超车的地方，也可以超自行车
  - 车站附近，可以超车
  - 隧道内多车道也可以超车
  - 人行横道30m以内，不能超车，也不能变道超车
- 雨天，停止距离会变成2倍
- 踏切
  - 如果没有红绿灯，必须一時停止
- 上下坡，坡顶 & 下坡附近不能超车
- 缓行
  - 优先道路没有缓行义务
- 步行者
  - 一般步行者保持安全距离就可以
  - 老人等特殊弱势群体，必须减速
  - 确定没有人，无论是人行横道还是安全地带，都可以正常通过
  - 如果不是人行横道，没有一時停止的义务
  - 如果是刚要出来并入主道，不管有没有人都要一時停止
- 内轮差：后轮在前轮曲线的内侧
- 自転車道 / 自転車横断帯
